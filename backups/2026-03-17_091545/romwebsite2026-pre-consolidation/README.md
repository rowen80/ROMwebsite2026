# romwebsite2026

Rebuild root for the new Ryan Owen Media website and request-form system.

## Purpose
This folder contains the new parallel rebuild project for the 2026 ROM website migration.

## Layout
- `docs/` - architecture notes, migration plans, field mappings
- `site/` - public website rebuild assets/code
- `form/` - request form and backend-related rebuild work
- `automation/` - n8n/export/flow notes and future automation assets
- `backups/` - local project backups or exports related to the rebuild
- `notes/` - scratch notes and working references

## Rules
- Treat this as the main root for the rebuild.
- Keep the live/current system separate.
- Prefer clear, boring names over clever ones.
- Preserve rollback paths.
